
# Campus Infrastructure Monitoring & Incident Management System

## Project Overview
This project is a backend microservice platform developed to monitor campus infrastructure and manage incident reporting. Users can report issues such as projector failures, lab unavailability, or network downtime, while administrators can track resource availability and update resolution status.

---

## Features
- Report campus infrastructure incidents
- View and update incident status
- Monitor resource availability
- REST APIs built using FastAPI
- Containerized using Docker
- CI/CD pipeline integration

---

## Tech Stack
- Python
- FastAPI
- Docker
- GitHub Actions / Jenkins
- DockerHub
